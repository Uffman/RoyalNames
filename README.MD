Simple C# console application that sorts strings with alfanumeric caracters.
First it sorts by names and than by roman number.
string[] names = new string[]
            {
                "Fillip XI",
                "Fillip IV",
                "Fillip III",
                "Fillipe I",
                "Henry II",
                "Henric II",
                "Henric IV"
            };
Output:           
    Fillipe I
    Fillip IV
    Fillip XI
    Fillipe I
    Henric II
    Henric IV
    Henry II
